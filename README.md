# SO-101 Gear Synchronized Parallel Gripper

Open-source gear synchronized parallel gripper for the SO-101 robotic arm.

## Features

- Gear synchronized jaws
- Single servo drive
- Parallel motion
- Centered gripping
- 3D printable
